
## 项目概述

**项目名称** 
你好，2025！

**目标**
用C++实现一个精美的跨年动画，包含两个烟花效果和文字展示。  
**技术栈**
C++。

```
大概我全用函数就行了吧，类与对象太难了。
```

### 第一幕

- 首先黑屏
- 5 闪烁出现 + 音效
- 4 闪烁出现 + 音效
- 3 闪烁出现 + 音效
- 2 闪烁出现 + 音效
- 1 闪烁出现 + 音效

### 第二幕：第一个烟花

- 火球从底部上升（抛物线轨迹）
- 爆炸瞬间，粒子向四周扩散
- 粒子减速，"再见，2025" 文字显现
- 文字保持，粒子继续下落并淡出
- 粒子完全消失，文字短暂停留


### 第三幕：过渡 

- 屏幕渐暗，星光闪烁加强
- 静谧的星空，准备下一个烟花


### 第四幕：第二个烟花 

- 火球从底部上升（不同轨迹）
- 更大规模的爆炸
- 粒子扩散，"你好，2026" 文字显现
- 文字保持，粒子华丽下落
- 文字闪烁强化


### 第五幕：大结局 

- 多个小烟花同时绽放
- 满屏庆祝效果
- 渐变到 "Happy New Year 2026!"
